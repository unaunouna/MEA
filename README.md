# Maturity Expectation Accountability: Age-Based Asymmetries in LLM-Mediated Child–Child Conflict

This repository contains the data and materials 
accompanying the paper submitted to AIES 2026.

## Contents

- `logs/` — Full session logs for all 1,600 
  sessions (400 per model: Claude, Gemini, GPT, 
  Grok) across four dataset conditions
- `prompts/` — System prompts for each model and 
  dataset condition (DS1–DS4)
- `scenario/` — Original scenario script in 
  English, Japanese, and Korean

## Models and Session Dates

- Anthropic Claude Sonnet 4.5 (Dec 6, 2025)
- Google Gemini 2.5 Pro (Dec 7, 2025)
- OpenAI GPT-5.1 (Dec 3, 2025)
- xAI Grok 4.1 (Dec 6, 2025)

## Notes

All sessions were conducted via consumer-facing 
interfaces in fresh conversation windows with 
memory and cross-session features disabled. Grok 
sessions employed pseudonyms (Mairu, Ayu) in place 
of role labels to prevent identity attribution 
errors. Session logs are provided as recorded; 
participant labels follow conventions described in 
the paper.
